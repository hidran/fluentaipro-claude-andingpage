# Branch `book/cap-10-1-mockups`

**Snapshot del progetto a fine § 10.1 — Visual Companion**

A questo stadio hai solo i **mockup HTML statici** generati dalla skill `visual-companion` di Superpowers. Sono iterazioni di design da aprire nel browser, **prima** di scrivere una sola riga di codice React.

## Come provarli

```bash
git clone -b book/cap-10-1-mockups https://github.com/hidran/fluentaipro-claude-andingpage.git
cd fluentaipro-claude-andingpage
open .superpowers/brainstorm/36729-1774475288/sections.html
```

Sono HTML+Tailwind autonomi (Tailwind via CDN). Nessuna build.

## File presenti

- `approaches.html` — 4 approcci stilistici proposti
- `hero-design.html` / `hero-design-v2.html` — iterazioni hero
- `features-design.html` / `features-design-v2.html` — iterazioni features

- `social-pricing.html` — variant con social proof
